MLNStyleAnnotation

 
@interface MLNStyleAnnotation : NSObject

An MLNStyleAnnotation represents a single point, line, or polygon shape that can be displayed on an MLNMapView within a geographical coordinate space.

Do not create instances of MLNStyleAnnotation directly. Instead, create an instance of MLNCircleStyleAnnotation, MLNLineStyleAnnotation, MLNPolygonStyleAnnotation, or MLNSymbolStyleAnnotation and use -[MLNAnnotationController addShape:] or -[MLNStyleAnnotationController addShapes:] to add an MLNStyleAnnotation to a map view.
