MLNAnnotationController

 
@interface MLNAnnotationController : NSObject

An MLNStyleAnnotationController is the object responsible for adding, removing, and updating MLNStyleAnnotations on a map.

Do not create instances of MLNStyleAnnotationController directly. Instead, create an instance of MLNCircleAnnotationController, MLNLineAnnotationController, MLNPolygonAnnotationController, or MLNSymbolAnnotationController and use -[MLNAnnotationController addShape:] or -[MLNStyleAnnotationController addShapes:] to add shapes to an MLNMapView.
